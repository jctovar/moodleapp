# Run Linting

Run ESLint to check code quality and find potential issues in:
- All TypeScript files in `src/**/*.ts`
- All HTML templates in `src/**/*.html`

This will report:
- Code style violations
- Unused variables
- Potential bugs
- Best practice violations
